Root

http://localhost:1234/api

Create new Hello : POST -> /hello

Content Type : application/json

Body :
{
"message" : "abc"
}

Update existing Hello : PUT -> /hello

Content Type : application/json

Body :
{
"id" : 1,
"message" : "abc"
}

Get all Hello : GET -> /hello

No body

Get single Hello : GET -> /hello/{id}

No body

{id} = 1

Delete Hello : DELETE -> /hello/{id}

No body

{id} = 1
